first open command prompt, type cd then copy and paste the directory of the overlay folder, should look like - (cd C:\Users\fraze\Downloads\Overlay1)

Next, type "pip install -r requirements.txt" and run it. this installs all the valid dependencies for the python script to work. 

after you get all your dependencies installed, run "python overlay.py" while you arent in the gunners sight and thermals are off. and boom! should work now!!

!!IMPORTANT!! - you must change your death text to "RETICLETEST3" for it to recognize it.

controls - 
same as normal game

"Y" to disable thermal color
"V" to scout (Plays "identified" crew sound)
